# TASKS

Agentic implementation plan for the DjangoAppliku Copier scaffold.

## Coding conventions (apply throughout all phases)

- PEP 8, 120-character line limit, double quotes, f-strings, `isort`
- Never use bare `except:` or `except Exception:` ŌĆö catch specific exceptions
- Use `logging` instead of `print()` for errors
- **After completing each phase: commit all changes before starting the next phase**
- Commit messages must be meaningful ŌĆö describe what changed, not just "phase N done"
- Format: short imperative subject line, blank line, then bullet points for detail

## Phase 3 ŌĆö Copier template: `copier.yml`

**Goal:** Define all template questions, defaults, conditional logic, and validators in `template/copier.yml`.

### Task 3.1 ŌĆö Create `template/copier.yml`

Define the following questions in order. Use Copier's `when` key for conditional questions.

```yaml
project_name:
    type: str
    help: "Human-readable project name (e.g. My App)"

project_slug:
    type: str
    help: "Python module name used in wsgi/asgi paths (e.g. my_app)"
    default: "{{ project_name | lower | replace(' ', '_') | replace('-', '_') }}"

python_version:
    type: str
    help: "Python version used in the Dockerfile base image (e.g. 3.12)"
    default: "3.12"

db_type:
    type: str
    help: "Database type to provision on Appliku"
    choices:
        - postgresql_18
        - postgresql_17
        - postgresql_16
        - postgresql_15
        - postgis_16_34
        - postgresql_16_pgvector
        - timescale_db_17
        - mysql_8
    default: postgresql_17

task_runner:
    type: str
    help: "Task runner (none / celery / huey)"
    choices:
        - none
        - celery
        - huey
    default: none

celery_broker:
    type: str
    help: "Celery message broker"
    choices:
        - redis
        - rabbitmq
    default: redis
    when: "{{ task_runner == 'celery' }}"

redis_version:
    type: str
    help: "Redis version to provision on Appliku"
    choices:
        - "8"
        - "7"
        - "6"
    default: "8"
    when: "{{ task_runner != 'none' and (task_runner == 'huey' or celery_broker == 'redis') }}"

use_beat:
    type: bool
    help: "Add a Celery beat scheduler process?"
    default: false
    when: "{{ task_runner == 'celery' }}"

media_storage:
    type: str
    help: "Media file storage backend"
    choices:
        - none
        - s3_compatible
        - volume
    default: none

email_backend:
    type: str
    help: "Email backend"
    choices:
        - console
        - smtp
        - sendgrid
        - mailgun
        - ses
    default: console

use_sentry:
    type: bool
    help: "Configure Sentry error tracking?"
    default: false
```

Rules:
- Do not add any questions beyond those listed
- `when` conditions must use Jinja2 string expressions exactly as shown
- All defaults must match the OVERVIEW

### Task 5.2 ŌĆö Create `scripts/regenerate_example.py`

```python
"""Wipe and regenerate the Copier-generated files in example/demo_project/."""
import shutil
import subprocess
from pathlib import Path

GENERATED_FILES = [
    "appliku.yml",
    "Dockerfile",
    "run.sh",
    "release.sh",
    "celery-worker.sh",
    "celery-beat.sh",
]

EXAMPLE_DIR = Path(__file__).parent.parent / "example" / "demo_project"
TEMPLATE_DIR = Path(__file__).parent.parent / "template"

COPIER_DATA = {
    "project_name": "Demo Project",
    "project_slug": "config",
    "python_version": "3.12",
    "db_type": "postgresql_17",
    "task_runner": "none",
    "media_storage": "none",
    "email_backend": "console",
    "use_sentry": "false",
}


def main() -> None:
    for filename in GENERATED_FILES:
        target = EXAMPLE_DIR / filename
        if target.exists():
            target.unlink()
            print(f"Removed {target}")

    data_args = []
    for key, value in COPIER_DATA.items():
        data_args += ["--data", f"{key}={value}"]

    subprocess.run(
        ["copier", "copy", str(TEMPLATE_DIR), str(EXAMPLE_DIR), "--defaults", "--overwrite"] + data_args,
        check=True,
    )
    print("Regeneration complete.")


if __name__ == "__main__":
    main()
```

## Phase 7 ŌĆö Phase 2 CLI: Appliku automation

**Goal:** Standalone CLI tool that reads Copier answers and automates Appliku provisioning via the REST API. This is a separate Python package, not a Django app.

### Task 7.1 ŌĆö CLI package structure

Create:
```
appliku_cli/
  __init__.py
  cli.py          ŌåÉ entry point
  credentials.py  ŌåÉ .env.appliku read/write
  api.py          ŌåÉ Appliku API client (thin wrapper around requests)
  provision.py    ŌåÉ orchestration logic
```

Register entry point in `pyproject.toml`:
```toml
[project.scripts]
appliku-setup = "appliku_cli.cli:main"
```

### Task 7.2 ŌĆö `credentials.py`

Responsibilities:
- Look for `.env.appliku` in the current working directory
- If missing, prompt user for `APPLIKU_API_KEY`, `APPLIKU_TEAM_PATH`, `APPLIKU_APP_ID` and write the file
- Add `.env.appliku` to `.gitignore` if not already present
- Return a `Credentials` dataclass

Never commit or log the API key value.

### Task 7.3 ŌĆö `api.py`

Thin wrapper around `requests`. Implement only the endpoints needed:

```python
class ApplikuClient:
    def __init__(self, api_key: str, team_path: str, app_id: int) -> None: ...

    def create_datastore(self, name: str, store_type: str) -> dict: ...
    # POST /api/team/{team_path}/applications/{app_id}/datastores
    # body: {"name": name, "store_type": store_type, "is_default": True}

    def set_config_vars(self, vars: dict[str, str]) -> None: ...
    # PATCH /api/team/{team_path}/applications/{app_id}/config-vars

    def trigger_deploy(self) -> dict: ...
    # POST /api/team/{team_path}/applications/{app_id}/deploy
```

Error handling:
- Raise a specific `ApplikuAPIError` on non-2xx responses ŌĆö include status code and response body
- Do not swallow errors silently

### Task 7.4 ŌĆö `provision.py`

Implement `run_provision(credentials, copier_answers)` following the Phase 2 setup sequence from OVERVIEW exactly:

1. Provision database datastore using `db_type` from `copier_answers`
2. If `task_runner != "none"` and broker is Redis: provision Redis datastore using `redis_version`
3. If `task_runner == "celery"` and `celery_broker == "rabbitmq"`: provision RabbitMQ datastore
4. If `media_storage == "volume"`: provision volume (via Appliku volumes API)
5. Generate `SECRET_KEY` with `secrets.token_urlsafe(50)` and push via `set_config_vars`
6. Prompt user for domain; push `ALLOWED_HOSTS` and `CSRF_TRUSTED_ORIGINS`
7. Push `WEB_CONCURRENCY` (prompt, default `"2"`)
8. If `media_storage == "s3_compatible"`: prompt for `AWS_ACCESS_KEY_ID`, `AWS_SECRET_ACCESS_KEY`, `AWS_STORAGE_BUCKET_NAME`, `AWS_S3_ENDPOINT_URL` and push
9. If `email_backend != "console"`: prompt for relevant `EMAIL_*` vars and push
10. If `use_sentry`: prompt for `SENTRY_DSN` and push
11. Trigger first deployment

Each step must log what it is doing (use `logging`, not `print`).

### Task 7.5 ŌĆö `cli.py`

Simple entry point:
- Parse `--copier-answers-file` (path to `.copier-answers.yml` written by Copier) or read from current directory
- Load credentials
- Call `run_provision`
- Print success message
